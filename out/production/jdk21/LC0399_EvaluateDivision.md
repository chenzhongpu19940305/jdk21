# LC0399 除法求值（Evaluate Division）

## 题目大意
给若干形如 `a / b = k` 的等式和若干查询 `x / y`，求每个查询的结果，若不可达返回 -1。

## 思路（图 + DFS/BFS）
- 将每个变量看作图中的节点；
- 对于等式 `a / b = k`：
  - 建边 `a -> b` 权重为 `k`；
  - 建边 `b -> a` 权重为 `1/k`；
- 对于查询 `x / y`：
  - 在图中从 `x` 出发，通过 DFS 或 BFS 搜索到 `y`，沿途累乘边权；
  - 若不可达则返回 -1。

## 复杂度
- 设变量数为 `V`，等式数为 `E`：
- 建图：O(E)。
- 每个查询最坏：O(V + E)。

## 考点
- 用图建模代数关系。
- DFS/BFS 搜索路径并累乘权重。
