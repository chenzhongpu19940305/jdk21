# LC0494 目标和（Target Sum）

## 题目大意
给定非负整数数组 `nums` 和整数 `target`，每个数前可以加 `+` 或 `-` 号，统计表达式结果为 `target` 的不同方案数。

## 思路（转化为子集和计数）
- 设所有数之和为 `sum`，把选为正号的子集和记为 `P`，负号子集和记为 `N`，则：
  - `P + N = sum`，`P - N = target`；
  - 联立可得：`P = (sum + target) / 2`；
- 问题转化为：有多少子集的和为 `P`。
- 典型 01 背包计数：
  - `dp[j]`：和为 `j` 的子集数目；
  - 初始化：`dp[0] = 1`；
  - 转移：对每个 `num`，`for j from P..num: dp[j] += dp[j-num]`。

## 复杂度
- 时间复杂度：O(n * P)。
- 空间复杂度：O(P)。

## 考点
- 将符号选择问题转为子集和问题。
- 01 背包的“方案数”写法。
