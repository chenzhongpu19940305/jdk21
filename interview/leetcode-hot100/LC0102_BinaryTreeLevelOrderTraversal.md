# LC0102 二叉树的层序遍历（Binary Tree Level Order Traversal）

## 题目大意
给定一棵二叉树，返回其节点值的层序遍历（从上到下，从左到右）。

## 思路（BFS 队列）
- 使用队列进行广度优先遍历：
  - 初始将根节点入队；
  - 每次处理一层：记录当前队列大小 `size`，循环 `size` 次：
    - 弹出队头节点，加入当前层结果；
    - 将其左右子节点（若非空）入队。
  - 完成一层后，将该层结果加入最终答案。

## 复杂度
- 时间复杂度：O(n)。
- 空间复杂度：O(n)。

## 考点
- BFS 在树上的典型用法。
- 分层遍历技巧（利用队列当前大小）。
