# LC0002 两数相加（Add Two Numbers）

## 题目大意
两个非空链表表示两个非负整数，数字以 **逆序** 方式存储，每个节点只包含一个数字。将这两个数相加，返回一个新的链表表示它们的和。

## 思路
- 本质是「按位相加 + 进位」：
  - 使用一个 `carry` 记录进位。
  - 从两个链表头开始，同时向后遍历：
    - 当前位的和 = `l1.val + l2.val + carry`。
    - 新节点值 = `sum % 10`，进位 = `sum / 10`。
  - 若某条链表已结束，视为该位为 0。
  - 最后如果还有进位 `carry > 0`，需要再补一个节点。

## 复杂度
- 时间复杂度：O(max(m, n))，m、n 为两个链表长度。
- 空间复杂度：O(1)（不计输出链表），只使用了常数级额外变量。

## 考点
- **链表基本操作**（指针移动、构造新链表）。
- **模拟加法运算与进位处理**。
- **边界情况**：
  - 链表长度不同；
  - 最后一位仍有进位需要新增节点。
