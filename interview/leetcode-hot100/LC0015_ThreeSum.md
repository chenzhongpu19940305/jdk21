# LC0015 三数之和（3Sum）

## 题目大意
给定整数数组 `nums`，判断数组中是否存在三个元素 `a, b, c`，使得 `a + b + c = 0`，找出所有满足条件且 **不重复** 的三元组。

## 思路（排序 + 双指针）
- 先对数组排序。
- 外层枚举第一个数 `nums[i]`：
  - 若 `nums[i] > 0`，后面都 >=0，不可能再有和为 0，直接 break。
  - 若 `i > 0` 且 `nums[i] == nums[i-1]`，跳过，避免重复三元组。
- 内层使用双指针在 `[i+1, n-1]` 上做「两数之和为 `-nums[i]`」：
  - 计算 `sum = nums[i] + nums[left] + nums[right]`。
  - `sum == 0`：记录答案，并跳过相同的 `left` 与 `right` 以去重。
  - `sum < 0`：`left++`。
  - `sum > 0`：`right--`。

## 复杂度
- 时间复杂度：O(n^2)。
- 空间复杂度：O(1)（不计输出）。

## 考点
- **排序 + 双指针** 的经典应用。
- 如何 **去重**（跳过相同的 `i`、`left`、`right`）。
