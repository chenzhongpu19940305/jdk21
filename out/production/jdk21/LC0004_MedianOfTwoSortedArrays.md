# LC0004 寻找两个正序数组的中位数（Median of Two Sorted Arrays）

## 题目大意
给定两个大小分别为 `m` 和 `n` 的 **有序（升序）数组** `nums1` 和 `nums2`，请你找出并返回这两个有序数组的 **中位数**，要求时间复杂度为 O(log(m+n))。

## 思路（划分 + 二分搜索）
- 不能直接合并后排序（O(m+n)），需要用 **二分** 降到 O(log(m+n))。
- 关键想法：
  - 在较短的数组 `nums1` 上二分一个切分点 `i`，在 `nums2` 上对应切分点为 `j`，满足：
    - `i + j = (m + n + 1) / 2`（左半部分长度固定）。
  - 保证：
    - `nums1[i-1] <= nums2[j]` 且 `nums2[j-1] <= nums1[i]`，
    - 此时左半部分最大值、右半部分最小值即可推中位数。
- 二分调整：
  - 如果 `nums1[i-1] > nums2[j]`，说明 `i` 取大了，`right = i - 1`。
  - 否则 `left = i + 1`。

## 复杂度
- 时间复杂度：O(log(min(m, n)))，只在较短数组上二分。
- 空间复杂度：O(1)。

## 考点
- **二分查找在答案空间 / 划分位置上的应用**（不是传统在值上二分）。
- **有序数组的中位数与“左右两部分长度相等”性质**。
- 边界处理：`i=0 / i=m / j=0 / j=n` 时的极值处理。
