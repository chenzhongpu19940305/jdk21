# LC0124 二叉树中的最大路径和（Binary Tree Maximum Path Sum）

## 题目大意
路径定义为二叉树中任意节点到任意节点的一条连续连接路径，路径上至少包含一个节点且不一定经过根。每个节点有正负权值，求所有路径中节点值之和的最大值。

## 思路（DFS + 贡献值）
- 对于节点 `node`：
  - 设从左子树能得到的最大「向上延伸路径和」为 `leftGain`，右子树为 `rightGain`：
    - 若子树返回值为负，则不如不要，因此 `leftGain = max(0, dfs(left))`。
  - 以 `node` 为 **最高点** 的一条路径，其和为：`node.val + leftGain + rightGain`。
  - 用全局变量 `maxSum` 维护所有节点作为最高点时的最大路径和。
  - 向父节点返回的值只能是一条向上的链，因此返回：`node.val + max(leftGain, rightGain)`。

## 复杂度
- 时间复杂度：O(n)。
- 空间复杂度：O(h)。

## 考点
- 树形 DP / 递归中维持全局最优。
- 对「路径」与「向上贡献」的区分。
