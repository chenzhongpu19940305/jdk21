# LC0053 最大子数组和（Maximum Subarray）

## 题目大意
给定一个整数数组，找到一个具有最大和的 **连续子数组**（至少包含一个元素），返回其最大和。

## 思路（Kadane 算法 / 动态规划）
- 设 `cur` 为「以当前元素结尾的最大子数组和」，`ans` 为全局最大：
  - 若 `cur < 0`，则当前子数组对后续只有拖后腿作用，应重新从当前元素开始：`cur = nums[i]`。
  - 否则继续累加：`cur += nums[i]`。
- 每一步更新 `ans = max(ans, cur)`。

等价 DP：`dp[i] = max(nums[i], dp[i-1] + nums[i])`。

## 复杂度
- 时间复杂度：O(n)。
- 空间复杂度：O(1)。

## 考点
- 线性 DP 思想。
- 最大子段和经典问题。
