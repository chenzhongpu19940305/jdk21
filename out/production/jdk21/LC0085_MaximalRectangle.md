# LC0085 最大矩形（Maximal Rectangle）

## 题目大意
给定一个只包含 0 和 1 的矩阵，找出其中只包含 1 的最大矩形，并返回其面积。

## 思路（每行转化为柱状图 + 单调栈）
- 将每一行都视作一个「底」，往上累计高度：
  - 遍历每行时，对每列 `j`：
    - 若当前是 `'1'`，则 `heights[j] += 1`；
    - 若是 `'0'`，则 `heights[j] = 0`。
  - 此时 `heights` 数组表示以当前行为底的柱状图高度。
- 对于每一行维护出的 `heights`，调用 **柱状图最大矩形**（LC0084）的解法。
- 取所有行的最大值即为答案。

## 复杂度
- 设矩阵大小为 `m x n`：
- 时间复杂度：O(mn)。
- 空间复杂度：O(n)。

## 考点
- 将二维矩阵问题转化为多次一维柱状图问题。
- 单调栈的综合应用。
