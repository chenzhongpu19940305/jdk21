# LC0003 无重复字符的最长子串（Longest Substring Without Repeating Characters）

## 题目大意
给定一个字符串 `s`，找出其中不含有重复字符的 **最长子串** 的长度。

## 思路
- 使用 **滑动窗口 + 记录上一次出现位置**：
  - 维护左右指针 `[left, right]` 表示当前无重复子串。
  - 使用数组/哈希表 `index[c]` 记录字符 `c` 上一次出现的下标。
  - 当 `right` 指向的字符 `c` 在窗口内出现过时，需要移动 `left`：
    - `left = max(left, index[c] + 1)`，确保窗口内无重复。
  - 每次更新答案 `res = max(res, right - left + 1)`。

## 复杂度
- 时间复杂度：O(n)，每个字符进出窗口最多一次。
- 空间复杂度：O(1) 或 O(字符集大小)，这里用定长数组 128 即可。

## 考点
- **滑动窗口思想**。
- **窗口内约束维护：无重复字符**。
- **字符上一次出现位置的记录和使用**。
