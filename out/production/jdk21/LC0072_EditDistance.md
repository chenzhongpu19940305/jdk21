# LC0072 编辑距离（Edit Distance）

## 题目大意
给定两个单词 `word1` 和 `word2`，计算将 `word1` 转换为 `word2` 所需的最小操作数。允许操作：插入、删除、替换一个字符。

## 思路（二维 DP）
- 定义 `dp[i][j]`：`word1[0..i-1]` 变为 `word2[0..j-1]` 的最小编辑距离。
- 初始：
  - `dp[i][0] = i`（全部删除）。
  - `dp[0][j] = j`（全部插入）。
- 转移：
  - 若 `word1[i-1] == word2[j-1]`：
    - `dp[i][j] = dp[i-1][j-1]`。
  - 否则，从三种操作中取最小：
    - 插入：`dp[i][j-1] + 1`；
    - 删除：`dp[i-1][j] + 1`；
    - 替换：`dp[i-1][j-1] + 1`。

## 复杂度
- 时间复杂度：O(mn)。
- 空间复杂度：O(mn)（可压缩为 O(min(m, n))）。

## 考点
- 经典 DP 字符串问题。
- 状态含义和三种编辑操作的建模。
