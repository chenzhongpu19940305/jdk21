# LC0437 路径总和 III（Path Sum III）

## 题目大意
给定一棵二叉树和一个整数 `targetSum`，统计和为 `targetSum` 的路径数目。路径必须向下（父到子），但可以从任意节点开始、任意节点结束。

## 思路（前缀和 + 哈希表）
- 定义根到当前节点的前缀和 `preSum`：
  - 对于当前节点，若此前存在某个前缀和 `pre`，使得 `preSum - pre = targetSum`，
    则以某个祖先节点之后开始，到当前节点结束的这段路径和为 `targetSum`；
- 使用哈希表 `prefixCount` 存储：`前缀和 -> 出现次数`：
  - 初始放入 `prefixCount[0] = 1`（空路径）；
  - DFS 时：
    - 先更新当前 `preSum`，统计 `preSum - targetSum` 在哈希表中的次数；
    - 然后将 `preSum` 计数 +1，递归左右子树；
    - 回溯时，将 `preSum` 计数 -1。

## 复杂度
- 时间复杂度：O(n)。
- 空间复杂度：O(n)。

## 考点
- 二叉树路径问题与前缀和思想的结合。
- DFS + 哈希表记录路径信息。
