# LC0079 单词搜索（Word Search）

## 题目大意
给定一个字符网格和单词 `word`，判断 `word` 是否存在于网格中。单词必须通过上下左右相邻的单元格顺序连接，不能重复使用同一个单元格。

## 思路（DFS + 回溯）
- 从每个格子作为起点进行 DFS：
  - 若当前字符不等于单词对应位置字符，失败返回。
  - 否则继续向四个方向扩展，查找下一个字符。
- 使用 `visited` 数组避免在一条路径中重复使用同一格子。
- 一旦某条路径完全匹配 `word`，立即返回 `true`。

## 复杂度
- 设网格大小为 `m x n`，单词长度为 L：
- 时间复杂度：最坏 O(mn · 4^L)。
- 空间复杂度：O(mn)（递归栈 + visited）。

## 考点
- 回溯法在二维网格中的应用。
- 搜索路径与访问标记的设计。
