# LC0283 移动零（Move Zeroes）

## 题目大意
给定一个数组，将所有 0 移动到数组末尾，同时保持非零元素的相对顺序。要求在原地操作、尽量减少操作步骤。

## 思路（双指针）
- 使用快慢指针：
  - `fast` 遍历数组；
  - `slow` 指向下一个要放非零元素的位置；
- 当 `nums[fast] != 0` 时：
  - 交换 `nums[slow]` 与 `nums[fast]`（或直接赋值），`slow++`。
- 遍历结束后，`slow` 前面的都是非零元素，后面自然是 0。

## 复杂度
- 时间复杂度：O(n)。
- 空间复杂度：O(1)。

## 考点
- 数组原地重排的双指针技巧。
