# LC0001 两数之和（Two Sum）

## 题目大意
给定整数数组 `nums` 和整数 `target`，在数组中找到两数之和等于 `target` 的下标，保证恰好存在一个解，同一元素不能使用两次。

## 思路
- 使用哈希表记录「值 -> 下标」。
- 遍历数组：
  - 当前值为 `x`，目标值为 `target - x`。
  - 如果哈希表中已经存在 `target - x`，说明找到了答案，直接返回两个下标。
  - 否则把当前值和下标存入哈希表中，继续遍历。

## 复杂度
- 时间复杂度：O(n)，每个元素只访问一次。
- 空间复杂度：O(n)，最坏情况下哈希表需要存所有元素。

## 考点
- **哈希表 / Map 的使用**。
- **时间复杂度优化：从 O(n²) 枚举变为 O(n)**。
- **数组遍历与下标操作**。
