# LC0022 括号生成（Generate Parentheses）

## 题目大意
给定 n 代表生成括号的对数，要求返回所有可能且合法的括号组合。

## 思路（回溯）
- 构造长度为 `2n` 的字符串，过程中保证任何前缀中左括号数量不少于右括号数量。
- 维护两个计数：
  - `open`：已经放置的左括号数量。
  - `close`：已经放置的右括号数量。
- 选择：
  - 若 `open < n`，可以放一个 `(`。
  - 若 `close < open`，可以放一个 `)`。
- 当长度达到 `2n` 时，得到一个合法方案。

## 复杂度
- 生成的答案数为第 n 个卡特兰数，时间主要消耗在输出这些答案上。
- 通常记为：时间复杂度 O(C_n)，空间复杂度 O(n)（递归深度）。

## 考点
- **回溯 + 剪枝**。
- 利用「任意前缀中左括号数 >= 右括号数」这一有效性条件。
