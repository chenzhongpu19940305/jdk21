# LC0337 打家劫舍 III（House Robber III）

## 题目大意
在一棵二叉树上，每个节点表示一间房屋的金额，相邻节点（父子）不能同时偷，求能偷到的最大金额。

## 思路（树形 DP，选与不选两种状态）
- 对于每个节点 `node`，返回两个值：
  - `dp[0]`：不偷当前节点时的最大金额；
  - `dp[1]`：偷当前节点时的最大金额。
- 状态转移：
  - 若偷当前节点：`dp[1] = node.val + left[0] + right[0]`；
  - 若不偷当前节点：`dp[0] = max(left[0], left[1]) + max(right[0], right[1])`。
- 最终答案为根节点 `max(dp[0], dp[1])`。

## 复杂度
- 时间复杂度：O(n)，每个节点遍历一次。
- 空间复杂度：O(h)，递归栈高度。

## 考点
- 树形 DP / 后序遍历。
- 为每个节点设计多状态返回值。
